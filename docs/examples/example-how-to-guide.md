# Как настроить расписание мастера

## Для кого
Владелец салона или администратор, который хочет задать индивидуальный график работы для мастера.

## Что понадобится
- Аккаунт администратора в Cita
- Мастер уже добавлен в систему (если нет — сначала добавьте мастера)

## Пошаговая инструкция

### Шаг 1: Откройте раздел "Расписание"
Зайдите в личный кабинет (admin панель) и нажмите **"Расписание"** в боковом меню.

> Вы увидите общее расписание бизнеса. По умолчанию все мастера работают по этому графику: Пн-Пт с 09:00 до 18:00, перерыв 13:00-14:00.

### Шаг 2: Выберите мастера
В верхней части страницы найдите выпадающий список **"Мастер"** и выберите нужного мастера.

> Если у мастера еще нет индивидуального расписания, вы увидите пустую таблицу. Это нормально — мастер пока работает по общему расписанию бизнеса.

### Шаг 3: Создайте расписание мастера
Нажмите кнопку **"Создать расписание мастера"**. Система скопирует текущее расписание бизнеса как основу.

### Шаг 4: Настройте дни недели
Для каждого дня недели вы можете:

1. **Включить или выключить день** — переключатель "Рабочий день". Если выключен, мастер не принимает записи в этот день.
2. **Задать время работы** — поля "Начало" и "Конец". Например, 10:00 - 20:00.
3. **Задать перерыв** — поля "Перерыв с" и "Перерыв до". Например, 14:00 - 15:00. Во время перерыва запись невозможна.

> Пример: мастер Алексей работает Вт-Сб с 10:00 до 20:00, перерыв 14:00-15:00. Воскресенье и понедельник — выходные.

### Шаг 5: Сохраните
Нажмите **"Сохранить"** после настройки каждого дня.

### Шаг 6: Проверьте результат
Откройте публичную страницу бизнеса и попробуйте записаться к этому мастеру. Доступные слоты должны соответствовать новому расписанию.

## Результат
Мастер теперь работает по своему индивидуальному графику. Клиенты при записи будут видеть только слоты, доступные в рамках этого расписания. Общее расписание бизнеса на этого мастера больше не влияет.

## Частые вопросы

**В: Что будет, если я не создам расписание для мастера?**
О: Мастер будет работать по общему расписанию бизнеса (Пн-Пт, 09:00-18:00 по умолчанию). Это подходит, если все мастера работают в одно время.

**В: Можно ли вернуться к общему расписанию?**
О: Да. Удалите индивидуальное расписание мастера (кнопка "Удалить расписание мастера"), и он снова будет работать по графику бизнеса.

**В: Как сделать выходной в конкретный день?**
О: Выключите переключатель "Рабочий день" для нужного дня недели. Например, если мастер не работает по понедельникам — выключите понедельник.

**В: Мастер работает в две смены (утро и вечер) с большим перерывом. Как настроить?**
О: Задайте общее время работы (например, 09:00-21:00) и перерыв на время "дырки" (например, 13:00-17:00). Система заблокирует слоты во время перерыва.

## Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| Клиенты видят старые слоты | Расписание обновляется сразу. Попробуйте обновить страницу (потянуть вниз в мобильном). Если не помогло — проверьте что вы сохранили изменения |
| Нет кнопки "Создать расписание мастера" | Убедитесь, что мастер выбран в выпадающем списке |
| Слоты во время перерыва все равно показываются | Проверьте что поля "Перерыв с" и "Перерыв до" заполнены корректно (время начала перерыва должно быть раньше времени окончания) |
| У мастера нет записей, хотя расписание настроено | Проверьте, что у мастера назначены услуги. Без услуг к мастеру нельзя записаться |
